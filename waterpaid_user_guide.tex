\chapter{Introduction}
Bienvenue dans le guide utilisateur de \textbf{WaterPaid}. Ce document détaille l'utilisation complète de l'application mobile, conçue pour révolutionner la gestion et la distribution de l'eau.

L'application se divise en deux interfaces distinctes :
\begin{itemize}
    \item \textbf{L'Interface Consommateur} : Pour les abonnés, permettant de suivre leur consommation, recharger leur crédit et gérer leur compteur.
    \item \textbf{L'Interface Administrateur (Superviseur)} : Pour les gestionnaires, offrant une vue globale sur le parc de compteurs, les utilisateurs et les transactions financières.
\end{itemize}

Ce guide vous accompagnera étape par étape à travers chaque fonctionnalité.

\chapter{Parcours Utilisateur (Vue Consommateur)}

Le parcours utilisateur couvre l'ensemble des interactions d'un abonné depuis sa première connexion jusqu'à la gestion complète de son compte et de sa consommation d'eau.

\section{Connexion et Authentification}

\subsection{Étape 1 — Connexion}
L'écran de connexion constitue le point d'entrée principal de l'application. L'utilisateur saisit son identifiant (numéro de téléphone ou email) ainsi que son mot de passe. 

À la validation, le système vérifie les identifiants de manière sécurisée et redirige automatiquement l'utilisateur vers son tableau de bord personnel.

\begin{figure}[H]
    \centering
    \includegraphics[width=0.4\textwidth]{captures/Interfaces/Login.png}
    \caption{Écran de connexion — Authentification sécurisée unique.}
    \label{fig:login_user}
\end{figure}

\subsection{Étape 2 — Création de Compte}
Si vous êtes un nouvel utilisateur, utilisez le formulaire d'inscription pour créer votre compte. Renseignez vos informations personnelles : nom, prénom, téléphone et mot de passe. Une fois validé, vous pourrez vous connecter immédiatement.

\begin{figure}[H]
    \centering
    \includegraphics[width=0.4\textwidth]{captures/Interfaces/CreerunCompte.png}
    \caption{Écran de création de compte — Inscription simple et rapide.}
    \label{fig:create_account}
\end{figure}

\section{Gestion Quotidienne}

\subsection{Étape 3 — Tableau de Bord Utilisateur}
Le tableau de bord est votre centre de contrôle. Il vous offre une vue synthétique et temps réel de votre situation :
\begin{itemize}
    \item \textbf{Jauge de Crédit} : Visualisez instantanément votre solde en Litres et en FCFA.
    \item \textbf{État de la Vanne} : Sachez si votre eau est ouverte ou coupée.
    \item \textbf{Consommation} : Suivez vos statistiques pour mieux gérer votre budget.
    \item \textbf{Actions Rapides} : Rechargez, liez un compteur ou consultez l'historique en un clic.
\end{itemize}

\begin{figure}[H]
    \centering
    \includegraphics[width=0.4\textwidth]{captures/Interfaces/DashboardUser.png}
    \caption{Tableau de bord utilisateur — Tout votre compte en un coup d'œil.}
    \label{fig:dashboard_user}
\end{figure}

\subsection{Étape 4 — Associer un Compteur (Lier)}
Pour utiliser le service, vous devez lier votre compte à un compteur physique WaterPaid.
\begin{enumerate}
    \item Munissez-vous du code d'association (Token) fourni par votre gestionnaire.
    \item Ouvrez le scanner QR Code depuis l'application.
    \item Scannez le code sur le compteur ou entrez le token manuellement.
\end{enumerate}

\begin{figure}[H]
    \centering
    \includegraphics[width=0.4\textwidth]{captures/Interfaces/CodeQR.png}

\end{figure}

\subsection{Étape 5 — Recharger son Crédit}
Ne soyez jamais à court d'eau grâce au module de paiement intégré :
\begin{enumerate}
    \item Choisissez un montant prédéfini ou saisissez une somme personnalisée.
    \item L'équivalent en volume d'eau est calculé automatiquement.
    \item Sélectionnez votre méthode de paiement (Mobile Money, Carte).
    \item Validez : votre vanne s'ouvre quasi-instantanément.
\end{enumerate}

\begin{figure}[H]

Gardez une trace de toutes vos dépenses. L'historique détaille chaque recharge avec la date, le montant et le volume d'eau obtenu, vous permettant une transparence totale sur votre consommation.

    \item \textbf{État du Parc} : Nombre de compteurs actifs, alertes batteries.
    \item \textbf{Graphiques} : Évolution des revenus en temps réel.
\end{itemize}

\section{Gestion du Parc (De l'Achat à l'Installation)}

Cette section vous guide pas à pas, de l'achat de votre matériel jusqu'à son installation chez le client.

\subsection{Étape Préalable — Acquisition du Compteur Physique}
Avant toute chose, en tant qu'Administrateur, vous devez vous procurer les compteurs d'eau intelligents WaterPaid.
\begin{enumerate}
    \item \textbf{Commande} : Contactez le service commercial WaterPaid ou votre distributeur agréé pour acheter des compteurs.
    \item \textbf{Réception} : Vous recevrez un carton contenant le compteur, sa vanne électronique et une carte contenant son Numéro de Série unique (ex: WP-123456).
    \item \textbf{Stockage} : Gardez ces compteurs en stock jusqu'à ce qu'un client demande un raccordement.
\end{enumerate}

\subsection{Étape — Installation et Gestion Physique du Compteur}
\textit{(Cette section sera complétée ultérieurement avec les détails techniques sur le vissage, la pose des joints et le raccordement à la tuyauterie).}
\vspace{2cm} % Espace pour le futur contenu
\noindent\rule{\linewidth}{0.4pt}

\subsection{Étape 3 — Inventaire et Création (Le Jumeau Numérique)}
Une fois le compteur physique en votre possession, vous devez l'enregistrer dans l'application pour le rendre "intelligent".
C'est ce qu'on appelle créer un "Compteur Virtuel" :
\begin{enumerate}
    \item Allez dans l'onglet \textbf{Compteurs} de l'application.
    \item Cliquez sur le bouton \textbf{"+ Nouveau Compteur"} (généralement en bas à droite).
    \item Entrez le \textbf{Numéro de Série} inscrit sur le boîtier du compteur physique.
    \item Validez : Le compteur apparaît désormais dans votre liste. Il est prêt à être lié !
\end{enumerate}

Accédez ensuite à la liste complète de vos dispositifs pour vérifier qu'il est bien "En ligne".

\subsection{Étape 5 \& 6 — Génération de Token (Attribution)}
Pour attribuer un compteur à un client :
1.  Générez un **Token d'Association** unique depuis la fiche du compteur.
2.  Le système crée un code temporaire sécurisé.
3.  Transmettez ce code au client (SMS, email, oral).
4.  Le client utilise ce code dans son application pour finaliser le lien (voir Partie Utilisateur, Étape 4).
